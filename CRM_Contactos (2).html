🔧 Conectar Microsoft 365 / Outlook

Para enviar emails directamente desde el CRM sin abrir Outlook, necesitas conectar tu cuenta Microsoft 365. Son 3 pasos que hace el administrador de tu cuenta una sola vez:
1
Registrar una app en Azure Active Directory
Ve a portal.azure.com → Azure Active Directory → Registros de aplicaciones → Nueva registro. Dale un nombre (ej: "CRM Contactos") y anota el Client ID y el Tenant ID.
2
Dar permisos de email
En la app registrada → Permisos de API → Agregar permiso → Microsoft Graph → Permisos delegados → añade Mail.Send y Mail.ReadWrite. Otorga consentimiento de administrador.
3
Introducir los datos en el CRM
Cuando tengas el Client ID y Tenant ID, dímelos y actualizo el CRM para que puedas autenticarte y enviar emails directamente desde aquí.
💡 Tip: Si no eres el administrador de Microsoft 365, reenvía estas instrucciones a tu departamento IT. Solo necesitan hacer esto una vez.
